View your app in AI Studio: https://ai.studio/apps/f770d6dd-e034-4ce2-83c1-be3d5cee6a6b

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
